# LDT Love Adventure RPG

Mini jeu navigateur multijoueur coop fantasy en vue du dessus.

## Lancer en local

```bash
npm install
npm start
```

Puis ouvre `http://localhost:3000`.

## Déploiement Render

- Build Command: `npm install`
- Start Command: `npm start`
- Root Directory: laisser vide si ce dossier est à la racine du repo
